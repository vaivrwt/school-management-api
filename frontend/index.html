Welcome to Educase School Management API

A clean API for adding school records and retrieving schools sorted by distance from a provided location.

Overview

Base route: /
API versioned routes: /api/v1/schools
Also available at root-level route aliases : /addSchool & /listSchools .

Endpoints

GET /health POST /addSchool { payload } GET /listSchools?latitude=<number>&longitude=<number> POST /api/v1/schools/addSchool { payload } GET /api/v1/schools/listSchools?latitude=<number>&longitude=<number>

Add School Payload

{ "name": "ABC School", "address": "Street, City", "latitude": 12.34, "longitude": 56.78 }

Typical Response

{ "success": true, "message": "School added successfully", "schoolId": 1 }
